Политика конфиденциальности для kototjom.top

Первая публикация: 29 ноября 2025 г.

Дата вступления в силу: 29 ноября 2025 г.

Настоящая Политика конфиденциальности описывает, как обрабатываются данные посетителей сайта kototjom.top (далее — «Сайт»).

1. Сбор и обработка данных

1.1. Данные, собираемые GitHub Pages

Сайт размещён на платформе GitHub Pages, которая управляется компанией GitHub, Inc. В рамках своей работы по обеспечению безопасности и стабильности платформы, GitHub автоматически собирает IP-адреса посетителей. Эти данные используются исключительно для защиты от DDoS-атак и предотвращения злоупотреблений. Я не имею доступа к этим данным и не могу их просматривать. Подробную информацию о сборе данных GitHub можно найти в Политике конфиденциальности GitHub.

1.2. Данные, собираемые сайтом

Я, как владелец Сайта, не собираю, не храню и не обрабатываю никакие персональные данные посетителей. На сайте нет форм для ввода данных, систем аналитики (таких как Google Analytics) или сторонних трекеров, которые могли бы собирать вашу личную информацию.

2. Файлы cookie

Сайт не использует собственные файлы cookie для отслеживания посетителей или хранения их данных.

3. Сторонние ссылки

Сайт может содержать ссылки на другие ресурсы в Интернете. Переходя по этим ссылкам, вы покидаете мой Сайт. Пожалуйста, имейте в виду, что политика конфиденциальности этих сторонних сервисов отличается от этой. Я не несу ответственности за их методы сбора и обработки данных. Рекомендую ознакомиться с их политикой конфиденциальности.

4. Контактная информация
